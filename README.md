Belajar HTML

1. Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun halaman web. HTML bekerja dengan menggunakan elemen-elemen (tags) untuk menentukan struktur dan konten halaman web.

Setiap elemen HTML memiliki:
Tag pembuka (misalnya <p>)
Isi elemen (misalnya teks dalam paragraf)
Tag penutup (misalnya </p>)

Beberapa elemen tidak memiliki tag penutup, seperti <img> dan <br>.

2. Struktur Dasar HTML

Berikut adalah struktur dasar sebuah dokumen HTML:
<!DOCTYPE html>  
<html lang="id">  
<head>  
    <meta charset="UTF-8">  
    <meta name="viewport" content="width=device-width, initial-scale=1.0">  
    <title>Judul Halaman</title>  
</head>  
<body>  
    <h1>Selamat Datang di HTML</h1>  
    <p>Ini adalah paragraf pertama saya!</p>  
</body>  
</html>  

Penjelasan Struktur:
<!DOCTYPE html> → Menentukan versi HTML yang digunakan (HTML5).
<html lang="id"> → Menandakan bahwa isi dokumen ini adalah HTML, dengan bahasa Indonesia.
<head> → Bagian kepala halaman yang berisi informasi metadata.
<meta charset="UTF-8"> → Menentukan karakter yang digunakan (UTF-8 untuk mendukung berbagai bahasa).
<meta name="viewport" content="width=device-width, initial-scale=1.0"> → Mengoptimalkan tampilan di perangkat seluler.
<title> → Menentukan judul halaman yang akan muncul di tab browser.
<body> → Bagian utama yang menampilkan konten halaman.
